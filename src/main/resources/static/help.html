Aplikacija se pokreće preko komandne konzole, naredbom: java -jar url-shortener.jar

Dostupni URL-ovi:

- localhost:8080/account (REST api, POST metoda, ulazni parametar: accountId)

- localhost:8080/register (REST api, POST metoda, ulazni parametri: url, redirectType)

- localhost:8080/statistic/{accountId} (REST api, GET metoda, path parametar: accountId)

- http://localhost:8080/showStatistics (web stranica za prikaz statistike)

- http://localhost:8080/short/{urlCode} (short url generiran metodom register)


REST metode se mogu testirati preko postmana.
Priložena je postman kolekcija za testiranje: URL Shortener.postman_collection.json
